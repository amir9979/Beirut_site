What is the diffrence between web 1.0/2.0/3.0?

-Web 1.0: Is the first generation of web and was mainly about spreading and sharing information.
   Web 1.0 was mainly read-only web and had little to no interaction with the clients.

-Web 2.0: Unlike the first generation the 2.0 web was a read-write web,giving the option
  to exchange information with the client thus enable more complex networking such as
   social media such as facebook or myspace.

-Web 3.0: Also known as the semantic web, web 3.0 unlike the previous versions uses
   semantics such as interpeting of queries or preforming computations
   to find the best products for you in shopping site (like Amazon recommendation).


Use case that shows the diffrences between web 1.0 to web 3.0

Lets take the example of "searching for a product online"
   Web 1.0 we need to search for the specific product if we want to find any information about it,unlike
   Web 2.0 we have the option of searching the product at social media.
   web 3.0 where we can give a description or some of the product name and the rest is done with semantics.
